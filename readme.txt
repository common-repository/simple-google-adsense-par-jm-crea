== Description ==

Simple Google Adsense par JM Créa permet d’afficher vos publicités Google Adsense sur n'importe quelles pages de votre site avec de simples shortcodes.  

Ce plugin vous offre la possibilité d'insérer 4 codes différents.

== Installation ==

1. Uploadez `simple-google-adsense-par-jm-crea` dans le dossier `/wp-content/plugins/` de votre site
1. Activez ’Simple Google Adsense par JM Créa’ depuis le menu « Extensions » de Wordpress
1. Rendez-vous ensuite dans l’onglet ‘Outils -> Simple Google Adsense’
